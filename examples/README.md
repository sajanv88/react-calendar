# React Calendar Example

This folder contains a working example of the React Calendar component.

## Running the Example

```bash
pnpm dev
```

This will start a development server at `http://localhost:3000` (or another port if 3000 is in use).

## What's Included

- **App.tsx**: Main example application showcasing the Calendar component
- **main.tsx**: Entry point for the React app

## Features Demonstrated

- ✅ Creating, editing, and deleting events
- ✅ Switching between Day, Week, Month, and Year views
- ✅ Event categories with color coding
- ✅ Dark mode support
- ✅ Drag and drop functionality
- ✅ Responsive design

## Customization

Feel free to modify `App.tsx` to explore different configurations and features of the calendar component.
